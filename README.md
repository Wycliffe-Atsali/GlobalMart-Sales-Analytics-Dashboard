# 📊 GlobalMart Sales Analytics Dashboard

### Transforming Retail Sales Data into Actionable Business Insights with Power BI

## Project Overview

The GlobalMart Sales Analytics Dashboard is an end-to-end business intelligence project developed using Microsoft Power BI. The project analyses retail sales data to uncover trends in revenue, profitability, customer behaviour, product performance, and regional sales.

Following the CRISP-DM methodology, the project transforms raw transactional data into interactive dashboards that support evidence-based business decisions. The dashboards are designed for executives, sales managers, and business analysts who require quick access to meaningful performance metrics and actionable insights.

## Business Problem

Retail organisations generate large volumes of sales data every day. However, without effective analysis, it is difficult to identify high-performing products, profitable customer segments, regional sales trends, and operational inefficiencies.

GlobalMart required a centralized analytical solution that would enable decision-makers to monitor business performance, identify growth opportunities, and make data-driven decisions using interactive visualizations.

## Project Objectives

The objectives of this project were to:

- Analyse historical retail sales performance.
- Identify revenue and profit trends.
- Evaluate product category performance.
- Analyse customer purchasing behaviour.
- Compare regional sales performance.
- Monitor key business KPIs.
- Develop an interactive executive dashboard for business decision-making

## 📂 Dataset

The analysis was conducted using a retail sales dataset containing transactional information for GlobalMart. The dataset includes sales records, customer details, product information, regional performance, and order information.

### Key Fields

- Order ID
- Order Date
- Customer ID
- Customer Name
- Product Category
- Product Sub-Category
- Product Name
- Sales
- Profit
- Quantity
- Discount
- Region
- Segment
- Ship Mode

The dataset was imported into Microsoft Power BI for cleaning, modelling, analysis, and dashboard development.

## 🔄 Project Methodology

This project followed the Cross-Industry Standard Process for Data Mining (CRISP-DM).

### 1. Business Understanding
Defined the key business questions related to sales performance, customer behaviour, profitability, and regional trends.

### 2. Data Understanding
Explored the dataset to identify data types, missing values, and key business attributes.

### 3. Data Preparation
Prepared the dataset in Power BI by checking data quality, validating fields, and preparing the data model for analysis.

### 4. Modelling
Developed calculated measures and KPIs using DAX and created relationships where necessary to support analysis.

### 5. Evaluation
Verified dashboard accuracy by comparing KPIs and visualisations against the underlying dataset.

### 6. Deployment
Published the interactive dashboard as a business intelligence solution for reporting and decision-making.
